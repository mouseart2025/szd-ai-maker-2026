# 需求文档：WiFi Web 控制应用

## 项目名称
手机 Web 控制 LED 和风扇

## 一句话描述
XIAO ESP32S3 作为 WiFi 热点，用户用手机连接后通过网页控制 Grove LED 和 Grove 风扇。

## 功能需求

### 功能 1：WiFi AP 热点
- XIAO 启动后自动创建 WiFi 热点
- 热点名称：`XIAO-Lab`（可在代码中修改为 `XIAO-Teacher01` 等避免冲突）
- 热点密码：`12345678`
- 串口输出热点 IP 地址（通常为 192.168.4.1）

### 功能 2：Web 控制页面
- 在 192.168.4.1 上提供一个 HTML 网页
- 网页包含以下控件：
  - **LED 开关按钮**：点击切换 Grove LED（D0）的亮灭状态
  - **风扇开关按钮**：点击切换 Grove 风扇（D1）的开关状态
  - **LED 当前状态显示**：显示"开"或"关"
  - **风扇当前状态显示**：显示"开"或"关"
- 网页设计要求：
  - 适配手机屏幕（响应式布局）
  - 按钮要大，方便触摸操作
  - 配色简洁，LED 按钮用绿色系，风扇按钮用蓝色系

### 功能 3：串口调试
- 每次操作在串口输出状态变化，如 "LED: ON"、"Fan: OFF"

## 硬件连接
- XIAO ESP32S3 Sense 通过 Grove Base 扩展板连接
- Grove LED 模块接在 D0 端口
- Grove 风扇模块接在 D1 端口
- USB-C 连接电脑（供电 + 调试）

## 使用方式
1. 上传程序到 XIAO
2. 手机打开 WiFi，搜索并连接 `XIAO-Lab`（密码 12345678）
3. 手机浏览器打开 `192.168.4.1`
4. 用网页按钮控制 LED 和风扇

## 技术要求
- 使用 PlatformIO + Arduino 框架
- 使用 `WiFi.h` 和 `WebServer.h` 库（ESP32 内置，无需额外安装）
- 串口波特率 115200
- HTML 页面内嵌在代码中（不使用外部文件）
